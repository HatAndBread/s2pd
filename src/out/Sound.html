Sound(source, volumeopt, loopopt, playbackRateopt)

Sound

Constructor

new Sound(source, volumeopt, loopopt, playbackRateopt)

Parameters:
Name Type Attributes Description
source string Source of audio file.
volume number <optional>
 Playback volume. An integer between 0 and 1. Default is 0.5
loop boolean <optional>
 Loop back to beginning of audio file on end? Default is false;
playbackRate number <optional>
 Speed of playback. 1 is nomrmal speed. 0.5 is half normal speed. 2 is twice normal speed etc. Changing playback rate will also affect pitch on most browsers. Default is 1.

s.enableAudio(); 
const backgroundMusic = new s.Sound('./someCoolMusic.mp3',0.5,true,1);
backgroundMusic.load();
button.onClick(()=>{
   backgroundMusic.play();
});

pauseButton.onClick(()=>{
  backgroundMusic.pause();
})
